Adding Selected Quiz Answers to an Array in JavaScript

    const chooseQuiz = (index, givenAns) => {
        const isExist = answers.find((ans) => ans.id === quizData[index].id);
        if (isExist) {
          let serial = 0;
          for (let quiz of answers) {
            if (isExist.id === quiz.id) {
              answers.splice(serial, 1, { ...quizData[index], givenAns });
              break;
            }
            serial++;
          }
        } else {
          answers.push({ ...quizData[index], givenAns });
        }
        displayAnswers(answers);
      };
The 'chooseQuiz' function in JavaScript adds selected quiz answers to an array. It takes two parameters, index and givenAns, which identify the quiz and its corresponding answer. The function checks if the quiz already exists in the answers array. If it does, the function updates the answer with the new value. Otherwise, it pushes a new object with the quiz and answer into the answers array. To call the function and display the answers, you need to pass the appropriate parameters to the function.
