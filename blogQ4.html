Stopping a JavaScript Timer with Conditional Statements

Bonnie Green • February 4th, 2023

4 min read • 1,570 views

In JavaScript, timers are often used to execute a piece of code after a specified amount of time has passed. However, sometimes we need to stop the timer before it completes its cycle. This can be achieved using conditional statements.

Here is an example of a function named quizTimer that stops a timer when a specific condition is met: 


                    const quizTimer = (dismiss) => {
                        if (dismiss) {
                          clearInterval(timer);
                          return;
                        }
                      }


This function takes a boolean parameter dismiss that determines whether to stop the timer or not. If dismiss is true, the function calls the clearInterval method to stop the timer.

To call the function and stop the timer based on a specific condition, you can pass a true value to the quizTimer function. For example, you can use an event listener to call the function when a button is clicked: 

<button onclick="quizTimer(true)">Stop Timer</button>


This code will stop the timer when the button is clicked.
