What are the differences between the functions map, filter, forEach, and find in JavaScript?

Ryan Lee • February 17th, 2023

2 min read • 1,570 views

map is a higher-order function that takes an array and applies a transformation function to each element, creating a new array with the transformed values.

filter is a higher-order function that takes an array and applies a predicate function to each element, creating a new array with only the elements that satisfy the predicate.

forEach is a higher-order function that takes an array and applies a function to each element, but it doesn't create a new array. It's used for performing side effects, like logging or updating a global variable.

find is a higher-order function that takes an array and applies a predicate function to each element, returning the first element that satisfies the predicate, or undefined if none do.

In summary, map transforms an array, filter creates a new array with only some of the original elements, forEach performs a function on each element without creating a new array, and find returns the first element that satisfies a condition.
